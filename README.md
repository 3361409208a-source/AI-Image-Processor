# AI 智能抠图工具 (rembg-cli)

这是一个基于深度学习的自动化抠图工具，专门用于精准去除图片背景。

## 目录结构
- **input/**: 放你需要处理的原始图片（支持 png, jpg, webp, bmp）。
- **output/**: 处理后的透明背景图片会保存在这里。
- **processor.py**: 核心处理逻辑。
- **run.bat**: Windows 一键执行脚本。

## 使用说明
1. 将你的原图放入 `input` 文件夹。
2. 双击运行 `run.bat`。
3. 等待命令行窗口提示完成后，去 `output` 文件夹查看结果。

## 依赖环境
此工具依赖 Python 3 和 `rembg` 库。
如果无法运行，请在命令行执行：
```bash
pip install rembg[cpu] pillow
```

## 技术原理
本项目使用 U2-Net AI 模型进行语义分割，能够智能识别物体轮廓，比传统的像素填充算法更精准，支持半透明边缘处理。
