# 🐟 AI 变异鱼工厂 (AI Mutant Fish Factory) V5.5

这是一套为游戏策划、原画师和独立开发者打造的 **全流程 AI 游戏资产流水线**。它集成了全球领先的生图算法与自动抠图技术，旨在将原本繁琐的素材制作流程缩减至秒级。

---

## 🚀 核心功能模块

### 1. ✍️ 文生图 (Text-to-Image)
*   **双引擎驱动**：支持 SiliconFlow (云端) 与 Stable Diffusion (本地) 模式。
*   **白底优化**：内置 Prompt 增强，专为输出干净、易抠图的素材设计。

### 2. 🎨 图生图进化 (Image-to-Image)
*   **草图进化**：上传手绘线稿或参考照片。
*   **属性继承**：在保持底图动作和轮廓的同时，由 AI 赋予其材质、光影与机械细节。

### 3. ✂️ 智能抠图 (Individual Removal)
*   **多精度引擎**：集成 `isnet` (高精)、`u2net` (全能) 等多种模型。
*   **透明 PNG 导出**：一键生成带透明通道的 RGBA 原始素材。

### 4. 🗂️ 批量生产线 (Batch Factory)
*   **工业化作业**：支持多文件同时上传。
*   **序列化产出**：自动排队处理并统一存入 `output/` 文件夹。

### 5. 🛒 全球风格广场 (Model Market)
*   **一门式搜索**：直接搜索 Hugging Face 全球数万款生图大模型。
*   **一键入库**：点击即可后台下载，自动关联本地实验室。

---

## 🏗️ 模块化技术架构 (V5.5)

本项目已完成深度重构，代码遵循高内聚低耦合原则：
- **`web_ui.py`**：交互指挥部。负责 Gradio 界面显示与用户指令调度。
- **`image_engine.py`**：核心生化室。封装 T2I/I2I 算法、Rembg 逻辑及显存优化。
- **`model_manager.py`**：物资储备中心。负责资产扫描、市场对接及存储管理。
- **`config_manager.py`**：系统资料室。负责 API Key 及全局配置的加密持久化。

## 🛠️ 安装与运行

### 环境要求
- Windows 10/11
- Python 3.10+
- 推荐 NVIDIA 显卡 (支持 CUDA 加速本地运算)

### 快速启动
1. 双击运行目录下的 `run_web.bat`。
2. 浏览器会自动打开 UI 界面。
3. 在“仓库与设置”中填入你的 API Key（如果使用云端模式）。

---

## 📂 目录说明
- `output/`：所有生成的透明素材原档均在此处。
- `config.json`：存储你的个人配置（请勿上传至公开 Git）。
- `~/.cache/huggingface/hub`：本地大模型仓库默认路径。
